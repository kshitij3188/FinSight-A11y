# How do I send a payment file?

**Source:** https://help.bunq.com/articles/how-do-i-send-a-payment-file
**Topics:** Payments

---

**Hey bunqers 🌈,**

With our new **Payment File feature** , you can execute a file with payments wherever you are, whenever you want.

Simply open the file with your bunq app (via email or attachment), confirm the transactions and execute the payments at once. No extra hardware needed!

## How does it work?

Please follow the steps below to execute payments from a **Payment File** :

  1. Open the file with the **bunq app**

  2. Tap **send payment**

  3. Click on **continue**

  4. Paid ✅

You can check out the file types that we support[ here](<http://www.sepaforcorporates.com/single-euro-payments-area/iso-20022-xml-in-less-than-2-minutes/>).

> 💡 To add a description or reference to the transfer, add it directly to the file. 

## How can I open an XML file with the bunq App on iOS? 

Just get the file onto your iOS device by: 

  * Airdropping the XML file to your iPhone 

  * Downloading the XML file from your Accounting App 

  * Opening the XML file in your Email App 

Then simply share the file and choose the option to open it with bunq. We’ll handle the rest by opening the file and processing the batch.

## How can I open an XML file with the bunq App on Android? 

Once the XML file is on your Android device phone, you can simply: 

  * Find the file and tap Share, then select the bunq app.

  * Or, tap the file and choose Open with > bunq.

However, it's good to know there's an exception with Gmail that doesn't allow opening of XML files.

**A quick heads-up:**

  * The default **My Files** app can be tricky with XML files for some Samsung users. If you run into trouble, downloading the **Google Files** app will let you open the XML file easily. 

  * The Gmail app won't let you open XML files directly. Just save the file to your phone first, then open it from there.

## Good to know

  * Available for **bunq Pro** and **bunq Elite** users

  * For paying large amounts additional authentication is always required, unless you've recently performed additional authentication within the same question.

  * Employees with full access to a sub-account belonging to their Business account can't choose it for a batch payment.

## FAQ

**Q: I made a mistake in my XML payment. Can you reverse it?**

A: Once a SCT payment is on its way, it’s final and can’t be reversed by us or any other bank. The best and quickest way to get your money back is to contact the recipient directly and ask them to return it.

To help you make sure the payment is always correct, we always show you a preview of your payments before you confirm. This gives you a chance to double-check that all the details are right before sending.

**Q: I've some XML files on my computer that I'd need to process. Is there a way to send them to my bunq app directly?**

A: Of course! You’ll just need to get the file onto your phone first. A simple way is to email it to yourself and open the attachment on your phone. From there, you can open it with the bunq app to process the payments.

**Q: Is it possible to schedule the XML batch to be paid in the future, instead of immediately when uploading?**

A: Right now, all payments from an XML file are processed immediately.
