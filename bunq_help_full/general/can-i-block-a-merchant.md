# Can I block a merchant?

**Source:** https://help.bunq.com/articles/can-i-block-a-merchant
**Topics:** Payments

---

**Hey bunqers 🌈,**

Do you want to block a merchant because they keep billing for an inactive subscription? In this article, we’ll see if this is possible and how to handle such cases yourself 🚀

## So, can I block a merchant?

Yes 🎉 You can easily block a merchant by following these steps:

  1. Go to 🏠 Home

  2. Tap on the desired Bank Account

  3. Select the card payment from a particular merchant

  4. Tap on the merchant’s icon

  5. Block Merchant

Another way to select from a list of merchants to block is the following:

  1. Tap on your Profile in the top left corner

  2. Tap on ⚙️ Settings

  3. Security

  4. Select the 🟥 Card Payments under "Blocked"

  5. Tap on “+ Add Merchant”

  6. Select merchant from the list

## What will happen if a blocked merchant tries to charge me?

If a blocked merchant tries charging you, the payment will be blocked. However, if you're expecting a refund from the blocked merchant, you'll be able to receive it without any problem 🎉

## Where can I review my blocked merchants?

You can get a clear overview of the merchants you’ve blocked by following the steps mentioned below:

  1. Tap on your Profile in the top left corner

  2. Tap on ⚙️ Settings

  3. Security

  4. Select the 🟥 Card Payments under "Blocked"

## How can I unblock a merchant?

To unblock a merchant, follow the steps below:

  1. Tap on your Profile in the top left corner

  2. Tap on ⚙️ Settings

  3. Security

  4. Select the 🟥 Card Payments under "Blocked"

  5. Select the merchant you want to unblock

  6. Tap “Unblock”

  7. Done ✅
