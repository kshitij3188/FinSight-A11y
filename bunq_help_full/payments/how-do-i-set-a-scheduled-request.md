# How do I set a Scheduled Request?

**Source:** https://help.bunq.com/articles/how-do-i-set-a-scheduled-request
**Topics:** Payments

---

Splitting subscriptions with friends? Set up a scheduled request and get paid automatically—no need to worry about reminders!

### How do I set a scheduled request?

  1. Go to the 🏠 Home tab

  2. Tap on the Bank Account you want to scheduled the request from

  3. Go to ⚙️ Settings

  4. Select **Scheduled Requests**

  5. Tap on **Create a Scheduled Request**

  6. Enter the amount and choose the recipient

  7. Select the date and frequency

  8. Confirm with your security code and you're done ✅

## How do I edit a scheduled request?

  1. Go to the 🏠 Home tab

  2. Tap on the Bank Account you scheduled the request from

  3. Go to ⚙️ Settings

  4. Select **Scheduled Requests**

  5. Tap on the scheduled request

  6. Select **Edit Scheduling**

## How do I cancel a scheduled request?

  1. Go to the 🏠 Home tab

  2. Tap on the Bank Account you scheduled the request from

  3. Go to ⚙️ Settings

  4. Select **Scheduled Requests**

  5. Tap on the scheduled request

  6. Select **Cancel Future Requests**
